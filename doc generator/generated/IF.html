IF
( a -- )
Types:
a
- float raw hex smart hex string

Tests the first item on the stack and skips to THEN if false.
Compile-only word

Example:
First, define a new word FOO:

: FOO DUP 20 > IF 1.1 * THEN ;



DUP makes a copy of the first item on the stack. 20 > compares the item to 20. If larger than 20, the value is multiplied by 1.1. If not, the steps between IF and THEN are skipped.



Related words: = <> > < ELSE THEN
